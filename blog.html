Question-1: What are the different ways to select an element in the DOM?

In the DOM, elements can be selected using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML includes HTML tags, innerText renders visible text only, and textContent includes all text, including hidden elements..

Question-3: What is event delegation in the DOM?

Event delegation is a technique in web development where you delegate event handling to a parent element rather than individual child elements, leveraging event bubbling to manage events more efficiently.

Question-4: What is event Bubling in the DOM?

Event bubbling in the DOM is a process where an event triggered on a child element propagates up through its ancestor elements in the hierarchy.
